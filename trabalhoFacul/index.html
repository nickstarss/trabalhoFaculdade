A Deep Web é uma parte obscura e misteriosa da internet que merece nossa compreensão. Diferente da Surface Web, que é facilmente acessível e indexada pelos motores de busca convencionais, a Deep Web consiste em páginas não indexadas, geralmente protegidas por senhas ou ocultas atrás de firewalls. Ela é usada por diversas razões legítimas, como sistemas de gerenciamento de banco de dados, intranets corporativas e redes privadas.

No entanto, também é conhecida como um espaço onde atividades ilegais podem ocorrer, devido ao anonimato que oferece aos usuários.

Definição:

Deep web e surface web conformam uma divisão do conteúdo da rede mundial de computadores (world wide web) quanto à indexação feita por mecanismos de busca padrão. Deep web (deepnet, web invisível, undernet, web obscura ou web oculta) corresponde à parte não indexada e surface web (ou internet superficial) é a parte indexada.
